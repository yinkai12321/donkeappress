# 默认模板

默认模板路径为 src\donk_layout\indexlayout\view\index-layout.vue

# 组件属性

属性 类型 说明
getList Function 查询列表方法
createEntity Function 新增方法
updateEntity Function 编辑方法
deleteList Function 删除方法
exportExcel Function 导出方法
importExcel Function 导入方法
getProperties Function 获取所有属性方法
columns Array 列配置
FormField Array 表单配置
FormData.sync Object 表单数据类
getListDetail Function 明细编辑列表查询方法
queryDetail Function 明细查询列表方法
columnsDetail Object 明细列
Config Object 配置项

# Config 配置项

配置项 类型 说明
dialog bool 表单是否为弹出框
toolbtn Object 工具栏按钮配置

# toolbtn 配置项

配置项 类型 说明
createBtn bool 是否显示新增按钮
updateBtn bool 是否显示编辑按钮
deleteBtn bool 是否显示删除按钮
importBtn bool 是否显示导入按钮
exportBtn bool 是否显示导出按钮

# 插槽

配置项 适用模板 说明
tablesearch 全部 自定义查询栏
toolbtn 全部 自定义工具栏
formbody 全部 自定义表单（使用前需注释fields.js 配置）
formtable 主从模板 自定义明细编辑
showdetail 主从模板 自定义明细展示
tablesearchdetail 主从模板 自定义明细查询栏
toolbtndetail 主从模板 自定义明细工具栏

注：主从模板可参考 src\wcs\WRouter\WCS_FlowControl\view\index.vue 注：单据型主从模板可参考 src\wms\InStock\Wms_StockInfo1\view\index.vue

# 示例

弹出框

示例代码

this.$DkDialogSelect({
    reqFunction: getMaterialsList(),
    selectType: "checkbox", // 单选radio  多选 checkbox
    columns: [
      {
        title: "工号",
        dataIndex: "ManNo",
        key: "ManNo",
      },
      {
        title: "人员姓名",
        dataIndex: "ManName",
        key: "ManName",
      },
      {
        title: "性别",
        dataIndex: "Sex",
        key: "Sex",
      },
    ],
    selectRow: (row) => {
      console.log("弹窗选择值", row);
    },
    confirm: (row) => {
      console.log("确定选择值", row);
    },
  });
